# Conceito

São operadores utilizados para *verificar se um objeto está presente em uma sequência*. Temos apenas *in* e *not in*.

```python
frutas = ['laranja', 'uva', 'limão']
print('laranja' in frutas)
>>> True

print('laranja' not in frutas)
>>> False
```
