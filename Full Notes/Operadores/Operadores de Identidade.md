# Conceito e Operadores

São operadores utilizados para *comparar se dois objetos testados ocupam a mesma posição na memória*. Temos apenas *is* e *is not*.

```python
curso = 'Python'
nome_curso = 'Python'
print(nome_curso is curso)
>>> True
```
