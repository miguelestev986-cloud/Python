# Conceito

São operadores utilizados para *comparar dois valores*. Eles retornam valores booleanos (True ou False). 

----

# Operadores

No Python temos:

1. == (É igual a)
2. != (Diferente de)
3. > (Maior que)
4. >= (Maior que ou igual a)
5. < (Menor que)
6. <= (Menor que ou igual a)
