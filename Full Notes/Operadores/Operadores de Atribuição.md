# Conceito

São operadores utilizados para *definir* o valor inicial ou *sobrescrever o valor de uma variável*.

----

# Operadores

1.  = (Atribuição simples)
2. += (Atribuição com adição)
3. -= (Atribuição com subtração)
4. /= (Atribuição com divisão)
5. //= (Atribuição com divisão inteira)
6. ** = (Atribuição com exponenciação)

```python
saldo = 1500
saldo += 200
#é como se estivéssemos dizendo 'saldo recebe saldo + 200' ou saldo = saldo + 200
print(saldo)
>>> 1700
```
