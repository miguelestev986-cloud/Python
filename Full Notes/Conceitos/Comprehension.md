# Conceito

A palavra "comprehension" em Python refere-se a uma sintaxe concisa e elegante para *criar estruturas de dados de coleção a partir de outras sequências ou iteráveis*. Elas são frequentemente mais *curtas*, mais *legíveis* e geralmente mais *rápidas* do que os laços tradicionais como o for.
