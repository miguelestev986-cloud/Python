# Conceito Geral

O laço while é usado para *repetir um bloco de código quando não sabemos o número exato de vezes* que nosso bloco de código deve ser *executado*.

```python
senha = '1234'
entrada = input('Qual a senha?')
while entrada != '1234':
	print('Senha incorreta! Tente novamente.')
	entrada = input('Qual a senha?')
```
