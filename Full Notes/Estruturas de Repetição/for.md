# Conceito Geral

O laço for serve para percorrer *elementos de uma sequência* (listas, strings...). Ele recebe cada item de um **iterável** e executa o bloco de código uma vez para cada item.

```python
for letra in 'Python':
	print(letra, end = ' ')
# O nome da variável não importa, pois ela apenas recebe o próximo valor fornecido pelo iterador interno.

>>>P y t h o n
```
