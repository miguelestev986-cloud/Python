# Conceito

Podemos usar métodos para analisar tuplas. Não podemos usar métodos para modificá-las como em outros tipos de coleção porque tuplas são imutáveis.

----

# Métodos

1. .count() é usado para determinar quantas vezes um objeto aparece dentro de uma lista;
2. .index() informa em que índice está a primeira ocorrência de um objeto;
3. len() informa o tamanho do objeto que foi passado como argumento;
```python 
tupla = ('Uva', 'Manga', 'Abacate',)
len(tupla) # 3
# Obs.: len é uma função built-in do python e não especificamente um método dessa classe
```
