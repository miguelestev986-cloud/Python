# Conceito

Funções são *blocos de códigos que executam papéis* dentro de um programa. Nós as criamos com **def**.

Por exemplo, se quisermos fazer um bolo: Precisamos *pegar* os ingredientes, *preparar* o forno (*Girar* o botão do fogão, *ligar* o fogo, *marcar* o tempo), *fazer* o preparo (*Peneirar* os secos, *bater* os líquidos, *adicionar* o fermento) e *assar*. Se precisarmos fazer bolo de diferentes sabores, reutilizar esse bloco de código tornaria o programa menos legível e manutenível, logo é *recomendável que façamos funções para cada ação que precisamos realizar de forma repetitiva*.
