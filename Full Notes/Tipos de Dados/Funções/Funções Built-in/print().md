# Para que serve

A função print é utilizada quando *queremos mostrar dados na saída padrão*.

----

# Parâmetros

Ela recebe um parâmetro do tipo *varargs* (ou seja, pode receber múltiplos argumentos) e quatro *parâmetros opcionais (sep, end, file e flush)*.
