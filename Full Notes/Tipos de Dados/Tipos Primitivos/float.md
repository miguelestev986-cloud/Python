# Conceito Geral

O tipo float (ponto flutuante) indica *valores reais*, como 0, 1.4, -192323...
Podemos *converter valores para o tipo float usando float()* e colocando o valor que será convertido dentro dos parênteses.

```python
n1 = float(input('Digite o primeiro número: '))
n2 = float(input('Digite o segundo número: '))
print(f'A soma vale {n1+n2})

# Se digitarmos 1 e 2, nos seria mostrado "A soma vale 3.0", por causa da conversão.
```
