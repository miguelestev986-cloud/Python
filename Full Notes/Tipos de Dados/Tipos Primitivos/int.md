# Conceito Geral

Int indica *valores inteiros*, como 1, 2 , -2943, 18938...
Podemos *converter um valor para de tipo inteiro usando int()* e dentro dos parênteses o valor que queremos converter.

```python
n1 = int(input('Digite o primeiro número: '))
n2 = int(input('Digite o segundo número: '))
print(f'A soma entre {n1} e {n2} vale {n1+n2})

# Aqui fazemos uma conversão explicíta do valor que input recebe e atribui à n1
```
