# Conceito

Podemos usar algum métodos para verificar os caracteres de uma string. Eles nos *retornam valores **booleanos*** como resultado da verificação.

# Métodos

1. .isnumeric() verifica se todos os caracteres na string são *numéricos*, mas *não aceita sinais de negativo ou ponto decimal.*;
2. .isalpha() verifica se é o valor da variável é "*alfabético*";
3. .isalnum() verifica se o valor da variável é *alfanumérico* (números e/ou letras);
4. .isspace() verifica se a string é *só espaços/brancos*;
5. .islower() verifica se todas as *letras estão em minúsculo*;
6. .isupper() verifica se todas as *letras estão em maiúsculo*.
