# Conceito Geral

O tipo bool (booleano) indica *valores lógicos* (Verdadeiro e Falso), que no Python são *True* (0) e *False* (1 ou qualquer outro valor diferente de 0)

```python
n = input('Digite um número: ')
print(bool(n))

# Se digitarmos um número, n terá valor e logo será True, se não digitarmos ela não terá valor e será False
```
