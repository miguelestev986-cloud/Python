# Conceito Geral

O tipo str (string)indica um valor de caractere ("Eu amo pudim", "Faltam 20 dias pro natal", etc). Podemos converter valores para o tipo str usando str() e colocando o valor dentro dos parênteses.

```python
num1 = 10
num2 = 20
print(num1 + num2)
print(str(num1) + str(num2))

>>>30
>>>1020
```
