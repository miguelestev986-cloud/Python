# Conceito Geral

Para criar uma estrutura condicional com *dois desvios*, podemos usar o **else** junto à **if**. Se a *condição testada* por if for *falsa*, o bloco de código dentro de *else será executado*.

```python
idade = int(input('Qual sua idade'))
if idade >= 18:
	print('Você é maior de idade.')
else:
	print('Você é menor de idade.')
```
